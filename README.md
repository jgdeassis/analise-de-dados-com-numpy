# Análise de Desempenho de Turma com NumPy

# Descrição

Este projeto tem como objetivo realizar a análise de desempenho de uma turma utilizando a biblioteca NumPy em Python, aplicando conceitos fundamentais de análise de dados e manipulação de arrays multidimensionais.

# Funcionalidades

* Cálculo da média geral da turma
* Análise do rendimento esperado da turma
* Estatísticas por prova (média, maior e menor nota)
* Estatísticas individuais por aluno
* Verificação de aprovação e reprovação com base na média

# Conceitos aplicados

* Arrays multidimensionais (NumPy)
* Operações vetorizadas
* Estatística básica (mean, sum, max, min)
* Manipulação de eixos (axis=0 e axis=1)
* Lógica condicional aplicada a dados

# Tecnologias utilizadas

* Python 3
* NumPy
